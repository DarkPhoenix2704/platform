## Getting Started

### Prerequisites

You need to install

1. [Node v16](https://nodejs.org/en/)
2. [pnpm](https://pnpm.io/)

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/tinkerhub/platform.git
   ```
2. Install NPM packages
   ```sh
   pnpm install
   ```
3. Start the dev server and open `http://localhost:3000`
   ```sh
   pnpm dev
   ```
